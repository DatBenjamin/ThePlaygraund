Proyectos Destacados

Dispositivo IoT de Monitoreo de Salud (HealthCare)

C++ | ESP32 | Web Server

Desarrollo de un sistema embebido basado en el microcontrolador ESP32 capaz de medir frecuencia cardíaca y saturación de oxígeno (SpO2) en tiempo real.

  • Implementación de Servidor Web HTTP interno en el chip para visualización remota de datos mediante HTML/CSS.
  • Programación en C++ gestionando protocolos de comunicación I2C y manejo de interrupciones.
  • Integración de sensores (MAX30100) y corrección de hardware (modificación de reguladores de voltaje) para compatibilidad lógica.

Agricultura de Precisión: Detección de Enfermedades en Plantas

Basado en el Convolutional Neural Networks (CNN). Es el estándar para visión por computadora para la detectecion de enfermedades de plantas.
